## About The Project

Linear Regression is one of the **most fundamental algorithms in Machine Learning**, but it is often treated as a black box using libraries.

In this notebook, I **build Linear Regression completely from scratch** to understand what actually happens behind `.fit()` and `.predict()`.

### What this project covers

- Understanding the **mathematical formulation** of Linear Regression  
- Implementing slope (**m**) and intercept (**b**) manually  
- Using **mean, variance, and covariance** concepts  
- Writing a custom regression class without using `sklearn`  
- Making predictions using the learned parameters  
- Gaining intuition behind **model fitting and error minimization**

This project focuses on **learning the internals**, not just using APIs.

## Getting Started

Follow these steps to run the notebook locally.

### Installation

1. **Clone the repository**:
```sh
git clone https://github.com/AadityaSinghWeb2005/Project-9-Linear-regression-From-Scratch.git
```

2. **Navigate to the project directory**:
```sh
cd Linear-regression-From-Scratch
```

3. **Launch Jupyter Notebook **:
```sh

jupyter notebook
```


Open the notebook and run cells sequentially.
